# 🎌 Anime Next Watch Generator

## 🚀 Project Overview
Anime Next Watch Generator is a web application designed for anime lovers who struggle to decide what to watch next. The app provides a curated list of trending anime and allows users to explore options based on their preferences such as genre or ratings.

The goal of this project is to simplify decision-making by offering quick, relevant, and engaging anime suggestions.

---

## 🎯 Features

- 🔥 **Trending Anime List**  
  Displays currently trending anime fetched from the API.

- 🎭 **Genre-Based Filtering**  
  Users can filter anime based on different genres.

- ⭐ **Top Rated Sorting**  
  Option to view anime sorted by ratings.

- 📖 **Expandable Synopsis Card**  
  Each anime includes a short description that can be expanded to read more.

- 🔍 **Search Functionality (Planned)**  
  Users will be able to search anime by name.

---

## 🌐 Public API Used

This project uses the **Jikan API**, an unofficial MyAnimeList API.

- API Documentation: https://docs.api.jikan.moe

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the application  
- **CSS3** – Styling and layout  
- **JavaScript (ES6)** – Logic and API handling  
- **React.js** – Frontend framework for building UI components
